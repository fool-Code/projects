Undergraduate research on Brazilian commuter networks

Python NetworkX Numpy Geopandas

Project Overview

As part of my undergraduate research, I modelled a commuter transport system as a graph: stations as nodes and routes as edges. Using Python + NetworkX, I measured how the average shortest path changes with network size (log of nodes vs. path length) and ran regressions to detect structural patterns.

This project taught me about data cleaning, graph theory metrics, spatial networks, and how to transform raw transport data into meaningful insights.

Technical Implementation

The commuter networks were treated as graphs where stations are nodes and routes represent weighted edges. The analysis focused on connectivity, shortest paths, and structural comparisons across different regions.

import networkx as nx
import numpy as np
from scipy.stats import linregress
import pickle

def build_graph(edge_list):
    G = nx.Graph()
    G.add_edges_from(edge_list)
    return G

def graph_stats(G):
    avg_path = nx.average_shortest_path_length(G)
    log_nodes = np.log(G.number_of_nodes())
    return log_nodes, avg_path

all_regions = [edges_region1, edges_region2, ...]

])

    logn_vs_avg[:,0], logn_vs_avg[:,1]
)
