Lattes Network Analysis

Undergraduate research on doctoral academic path networks

Python NetworkX Pandas

Project Overview

In this study I explored the Brazilian Lattes Platform data to understand the movement of PhD researchers across institutions and states. Using public CV information, I created visualisations to show patterns such as the fraction of updated CVs, the number of PhD holders by area and region, and the difference between funded and non-funded PhD scholarships.

The aim was to reveal how doctoral training is spread in the country and how researchers build their academic path.

Technical Implementation

The analysis was done fully in Python with libraries like pandas, networkx and matplotlib. I cleaned and filtered the data, then built a directed network where nodes represent institutions and edges show PhD transitions. Edges were weighted by the number of researchers moving between two institutions.

With this graph I calculated degree distributions, highlighted the main hubs, and produced cumulative plots in log scale to check network structure. This workflow helped me to transform large raw data into clear figures and insights about the Brazilian academic network.

Python Analysis Code

import pandas as pd
import networkx as nx
import numpy as np
import matplotlib.pyplot as plt
import pyhvr

# Prepare and filter data
df_filtered = df_author_filtered_phdinst_not_nan.drop(columns='name')
edge_list = df_filtered.groupby(['phd_institution_fixed','institution_fixed'])['id'].count().reset_index()
edge_list.columns = ['source','target','weight']
edge_list = edge_list.sort_values('weight',ascending=False)

# Create directed graph
G = nx.from_pandas_edgelist(edge_list[edge_list.weight>15], edge_attr='weight', create_using=nx.DiGraph())

# Degree analysis and cumulative distribution
degrees = dict(G.degree())
plt.loglog(*pyhvr.cdf(np.array(list(degrees.values())), log=True, nbins=25).T, '-o')
plt.xlabel('Degree')
plt.ylabel('Cumulative Distribution')
plt.tight_layout()
plt.show()

# Highlight top hubs in the network
threshold = 10
hubs = [n for n, d in degrees.items() if d >= threshold]
node_colors = ['#af8dc3' if n in hubs else '#5dade2' for n in G.nodes()]
nx.draw_networkx(G, node_color=node_colors, with_labels=False)
plt.show()
