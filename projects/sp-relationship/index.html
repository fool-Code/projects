Supernatural Network

A small network visualisation

Python NetworkX BeautifulSoup

Project Overview

In this project, I explored the relationships between characters in the popular TV show Supernatural. Using web scraping techniques, I collected relationship data from the Supernatural Fandom Wiki and visualised it as network graphs. The goal was purely exploratory: to understand character interactions and the structure of their connections. Having watched all the seasons, I was always curious to find out which character had the most romantic relationships. Naturally, I expected it would be Dean, given his reputation on the show. This curiosity inspired me to explore the data and visualise the connections between characters. While I didn't aim for predictive outcomes, the project helped me learn practical skills in web scraping, data cleaning, and network analysis.

Technical Implementation

This project analyses romantic relationships in the TV series Supernatural by scraping character data from the Supernatural Fandom wiki and visualising it as a network graph. The workflow includes web scraping, data cleaning, and building an interactive graph using Python and D3.js.

Process Overview

Fetch HTML with Requests → Parse with BeautifulSoup → Build Pandas DataFrame → Create NetworkX Graph → Visualise with Matplotlib or D3.js
