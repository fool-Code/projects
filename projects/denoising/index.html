Pytorch Python imageio

Project Overview

For this project I wanted to try PyTorch for image denoising, more as a curiosity than for any serious outcome. I built a small model using Gated Attention Units (GAU) to remove noise from CIFAR-10 and True Color Kodak images. The idea was simple: add Gaussian or Poisson noise to clean images, train a network to recover the original ones, and check how much it can clean them.

I started this project mostly by exploring code suggested by a ChatGPT model. At first I didn’t fully understand every part, but while debugging and fixing small errors I learnt a lot about PyTorch, from custom datasets to building and training a simple network. This experience then led me to my most recent project: studying PyTorch more systematically using the book Deep Learning with PyTorch and taking an online course in Mathematical Statistics.

PyTorch Denoising Code

import torch
import torch.nn as nn
from torchvision import datasets, transforms
from pytorch_msssim import ssim

def add_gaussian_noise(img, std=0.1):
    noise = torch.randn_like(img) * std
    noisy_img = img + noise
    return torch.clamp(noisy_img, 0., 1.)

class NoisyCIFAR10(torch.utils.data.Dataset):
    def __init__(self, train=True, noise_type="gaussian"):
        self.dataset = datasets.CIFAR10(root="./data", train=train, download=True, 
        transform=transforms.ToTensor())
        self.noise_type = noise_type

    def __len__(self):
        return len(self.dataset)

    def __getitem__(self, idx):
        clean_img, _ = self.dataset[idx]
        if self.noise_type == "gaussian":
            noisy_img = add_gaussian_noise(clean_img.clone())
        else:
            noisy_img = clean_img
        return noisy_img, clean_img
